Claude Usage Widget

A floating macOS widget that tracks your Claude usage limits in real time. Know when you're about to get rate-limited.

Install (macOS 13+)
brew install --cask rishi-banerjee1/ai-tools/claude-usage-widget
Or download from GitHub Releases

What It Does

🔴

Live Usage %

Progress ring with color-coded pace tracking: green, orange, red

⏱

Reset Countdown

Know exactly when your limits reset. No more guessing.

💠

Compact Mode

Double-click to shrink to a tiny frameless ring on your desktop

Widget States

On track   |   Compact   |   Still usable   |   Session expired

Why This Exists

  • Claude's rate limits are invisible until you hit them
  • The web UI doesn't show your usage prominently
  • No notification when you're close to the limit
  • This widget keeps it visible on your desktop at all times
